What are Tags and Attributes?

Tags and attributes are the basis of HTML. They work together but perform different functions – it is worth investing 2 minutes in differentiating the two.

What Are HTML Tags?

Tags are used to mark up the start of an HTML element and they are usually enclosed in angle brackets. An example of a tag is: < h1 >
Most tags must be opened < h1 >and closed < h1 > in order to function.

What are HTML Attributes?

Attributes contain additional pieces of information. Attributes take the form of an opening tag and additional info is placed inside. An example of an attribute is: >< img src="mydog.jpg" alt="A photo of my dog." > In this instance, the image source (src) and the alt text (alt) are attributes of the < img > tag.

Few Basic Tags of HTML

< !DOCTYPE > : It is an instruction that signals the browser about the markup language in which the current page is written.
< html > : It is the root of the HTML document as it is the container of all other HTML elements.
< head > : It contains the metadata of the HTML document.
< meta > : It defines the metadata of the HTML document.
< body > : It defines the document body and contains all the content of the HTML document.
< title > : It is used to define the document’s title.
< img > : It is used to add an image in an HTML document.
< h1 to h6 > : These tags are used to define headings.
< p > : It is used to define paragraphs.
< br > : It is used to insert a single line break in a document.
< ul > : It is used to define an unordered list.
< li > : It is used to define a list item.
< a > : It is used to define a hyperlink which is used to link one page to another.
< div > : It defines a division or a section in an HTML document.
< nav > : It is used to provide the navigation links within the current document or to other documents.
< header > : It contains the introductory elements of the HTML documents such as headings.
< footer > : It is used to define the footer section of an HTML document.
